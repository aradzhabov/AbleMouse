================================================================================
ДОКУМЕНТАЦИЯ ПО НАСТРОЙКАМ config.json для AbleMouse AI edition
================================================================================

[РАЗДЕЛ: communication] - НАСТРОЙКИ СВЯЗИ С СЕРВЕРОМ
--------------------------------------------------------------------------------
use_menu_system          : true/false - Использовать интеграцию с AbleMouse Beyond Switch
                           Если false - программа работает автономно без меню

menu_host                : "localhost" или IP-адрес - Адрес сервера меню

menu_port                : 12345 - Порт для подключения к серверу меню

================================================================================
[РАЗДЕЛ: main] - ОСНОВНЫЕ НАСТРОЙКИ
--------------------------------------------------------------------------------
camera                   : 0 - встроенная камера
                           1,2,... - внешняя камера (номер устройства)

cam_mouse_control        : true/false - Включить/выключить управление курсором через камеру

display_fps              : true/false - Отображать счетчик FPS на экране
                           (может немного снижать производительность)

do_not_move_cursor_if_xy_move_within_threshold : true/false - Пропускать микро-движения
                           Если true - курсор не двигается при очень маленьких движениях

skip_x_threshold         : 0.001 - Порог пропуска по оси X (меньше = чувствительнее)

skip_y_threshold         : 0.001 - Порог пропуска по оси Y (меньше = чувствительнее)

nose_center              : true/false - Центрировать курсор по носу
                           true - центр экрана соответствует носу
                           false - нос смещен относительно центра

last_action_display_time : 3.0 - Время отображения последнего действия на экране (секунды)

================================================================================
[РАЗДЕЛ: eye_and_mouth] - НАСТРОЙКИ ОТСЛЕЖИВАНИЯ ГЛАЗ И РТА
--------------------------------------------------------------------------------
left_eye_close_time_threshold       : 1.5 - Время закрытия ЛЕВОГО глаза для переключения
                                      режима управления (секунды)
                                      Пример: закройте левый глаз на 1.5 сек для
                                      включения/выключения управления курсором

right_eye_right_click_time_threshold : 1.0 - Время закрытия ПРАВОГО глаза для правого клика
                                      (секунды)

mouth_open_right_click_time_threshold : 1.0 - Время удержания рта открытым для правого клика
                                        (секунды)

mouth_open_menu_selection_threshold   : 0.3 - Время открытия рта для выбора пункта меню
                                        (секунды)

eye_switch_cooldown_duration          : 1.0 - Задержка после переключения режима (секунды)
                                        Чтобы случайно не переключить дважды

right_eye_right_click_cooldown_duration : 1.0 - Задержка после правого клика правым глазом
                                         (секунды)

mouth_right_click_cooldown_duration   : 1.0 - Задержка после правого клика ртом (секунды)

mouth_click_cooldown_duration         : 0.5 - Задержка после обычного клика ртом (секунды)

mouth_menu_selection_cooldown_duration : 0.8 - Задержка после выбора меню ртом (секунды)

eye_close_threshold    : 0.005 - Порог определения закрытого глаза
                         МЕНЬШЕ = чувствительнее (глаз легче считается закрытым)
                         БОЛЬШЕ = менее чувствительно (нужно сильнее закрыть глаз)

mouth_open_threshold   : 0.004 - Порог определения открытого рта
                         МЕНЬШЕ = чувствительнее (легче регистрируется открытый рот)
                         БОЛЬШЕ = менее чувствительно (рот нужно открывать шире)

================================================================================
[РАЗДЕЛ: filtering] - НАСТРОЙКИ ФИЛЬТРАЦИИ И СГЛАЖИВАНИЯ
--------------------------------------------------------------------------------
filter_method           : "smooth" или "kalman" - Метод сглаживания движений
                         "smooth" - экспоненциальное сглаживание (рекомендуется)
                         "kalman" - фильтр Калмана (в разработке)

smoothing_alpha         : 0.5 - Коэффициент сглаживания (только для метода "smooth")
                         0.0 - максимальное сглаживание (очень плавно, но с задержкой)
                         0.5 - среднее сглаживание (рекомендуется)
                         1.0 - без сглаживания (максимальная отзывчивость, но дрожание)

move_threshold_pixels   : 2 - Минимальное перемещение в пикселях для обновления курсора
                         Меньше = более плавное движение, но больше нагрузки на процессор
                         Больше = курсор менее чувствителен к микро-движениям

use_movement_threshold  : true/false - Включить/выключить порог движения

use_median_filter       : true/false - Использовать медианный фильтр для подавления шума

median_filter_window    : 10 - Размер окна медианного фильтра (количество кадров)
                         Больше = более плавно, но больше задержка

================================================================================
СОВЕТЫ ПО НАСТРОЙКЕ:
================================================================================
1. Если курсор сильно дрожит:
   - Увеличьте smoothing_alpha (0.6-0.8)
   - Увеличьте median_filter_window (15-20)
   - Увеличьте move_threshold_pixels (3-5)

2. Если курсор медленно реагирует:
   - Уменьшите smoothing_alpha (0.3-0.4)
   - Уменьшите median_filter_window (3-5)
   - Установите use_movement_threshold = false

3. Если глаза/рот плохо распознаются:
   - Уменьшите eye_close_threshold (0.003-0.004)
   - Уменьшите mouth_open_threshold (0.002-0.003)

4. Если случайные срабатывания:
   - Увеличьте *_cooldown_duration значения
   - Увеличьте пороговые значения
