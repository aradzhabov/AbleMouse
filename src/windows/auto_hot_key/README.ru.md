> Представьте ситуацию: вы можете управлять курсором, 
но только нестандартными способами (например с помощью глаз) — БЕЗ классической мыши, клавиатуры и других привычных устройств ввода. 
Встроенные средства Windows в таких условиях оказываются не слишком удобными. 
Например, чтобы переместить курсор из одного угла экрана в противоположный, вам всё равно придётся «вести» его по всему пути — быстрого способа «телепортации» курсора нет.
При этом в системе есть множество полезных сочетаний клавиш и вспомогательных утилит, которые могли бы значительно упростить задачу. 
Но как их вызвать, если вы не можете использовать клавиатуру или другие стандартные методы ввода?

Скрипт автоматизации на базе жестов, который распознает движение мыши (например, влево → вправо) как пример для вызова настраиваемой панели быстрого доступа к различным системным функциям.



Этот подход был разработан для людей с ограниченными возможностями, чтобы упростить и ускорить рабочий процесс в Windows. Особенно полезен для тех, кто может управлять только указателем мыши и не пользуется клавиатурой.

## Базовые возможности
- Распознавание жестов мыши (например, движение Влево → Вправо)
- Быстрый вызов экранной клавиатуры (OSK) и её перемещение к курсору
- **mouse jump**: Быстрый переход в любую точку экрана
- **crosshair**: Режим визуального прицела (перекрестие)
- Легкое добавление новых функций через AutoHotkey

## Установка
1. Установите [AutoHotkey v2.0](https://www.autohotkey.com)
2. Установите [PowerToys 0.94+](https://github.com)
3. Запустите [mouse_commander.ahk](./mouse_сommander.ahk)

## Использование
- Выполните жест **ВЛЕВО → ВПРАВО** (амплитуда > 50px, время < 1с).
- В появившемся меню выберите нужное действие.
- Меню автоматически закроется через 3 секунды, если не было нажатий.

## Конфигурация
В коде скрипта можно настроить:
- Чувствительность жестов (`minDistance`, `gestureTimeWindow`)
- Горячие клавиши для вызова функций
- Цвета и прозрачность интерфейса
